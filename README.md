# Browser Launcher

A simple Python-based GUI app that lets you quickly access download links for 20 different web browsers.  
Includes browser logos, hover effects, and rounded UI corners.

## Features

- Quick access to official browser download links
- Beautiful GUI with logos
- Hover effects and rounded corners
- Works on Windows (requires the included `.exe`)

## How to Use

1. Download the `BrowserLauncher.zip` from this repository.
2. Extract the folder if it’s zipped.
3. Double-click `browser_browser.exe` to run the app.
4. Click the download buttons to open official browser download pages.

## Requirements

- Windows 10 or higher
- No installation required if using the `.exe`

## Notes

- If the `.exe` doesn’t run, Windows may block unsigned programs. Right-click → Run as administrator or scan with antivirus.
